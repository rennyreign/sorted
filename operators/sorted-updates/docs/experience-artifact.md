# Sorted Updates Experience Artifact

A business owner sends a short WhatsApp message to Sorted Updates:

```txt
Can you add a beginners page to the site?
```

The owner should not feel like they are chatting with a general assistant. The experience should feel like sending a clear maintenance request to a reliable operator who already knows the site.

The system replies in a short, practical style:

```txt
Sorted - I can add a beginners page using the existing site style. I'll prepare the update as a preview first.
```

Behind the scenes, the WhatsApp Intake normalizes the message, the Client Router sends it to the GB Halesowen Operator, and the operator checks the request against the approved update menu. If the request touches pricing, legal, payments, medical claims, integrations, brand identity, or deletion, the operator stops and asks for manual approval.

The intended feel is constrained, fast, specific, and low-pressure.
