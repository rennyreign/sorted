# Build Notes

## What worked well

The mockup had a clear local-business conversion structure, so the build could stay close to the approved direction while improving spacing, responsive behaviour, and asset treatment.

## What slowed the build down

No official logo file was provided, so the header uses a lightweight CSS/SVG-style brand mark instead of a real logo asset.

## Reusable patterns

- Top contact bar plus sticky nav for local service sites.
- Five-card programme row that collapses to two columns and then one column.
- Contact section combining map-like location panel and direct booking CTA.

## Reusable components

- Gracie Barra-style local brand header treatment.
- Benefit strip with icon columns.
- Timetable prompt that avoids inventing exact class times.

## Copy patterns worth reusing

- "Book a free intro"
- "Find the right class before you step on the mat"
- "Classes for beginners, kids and competitors"

## Premium upgrade ideas

- Add official logo and brand assets.
- Add real coach profiles and photography.
- Add animated class timetable with filters.
- Add real Google review embeds or verified quotes.

## Things to avoid next time

- Do not invent exact timetable slots unless the client confirms them.
- Do not over-style the academy into a startup or SaaS look.

## Future automation opportunities

- Standardise a martial arts / gym one-page template using the programme, benefits, kids, testimonials, and contact pattern.
