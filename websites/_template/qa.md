# Website QA Checklist

## Conversion

- [ ] Primary CTA visible above the fold
- [ ] Contact path obvious
- [ ] Mobile CTA easy to access

## Content

- [ ] No lorem ipsum
- [ ] No placeholder copy
- [ ] Copy sounds human and local
- [ ] Services/products are clear

## Design

- [ ] Layout matches approved direction
- [ ] Mobile layout checked
- [ ] No horizontal scrolling
- [ ] Images cropped correctly
- [ ] Typography hierarchy clean
- [ ] Animations smooth and restrained

## Technical

- [ ] Metadata added
- [ ] Alt text added
- [ ] Footer links added
- [ ] No dead links
- [ ] No obvious console errors
- [ ] Responsive behaviour verified

## Final

- [ ] Site feels owner-ready
- [ ] Premium treatment consistent if applicable
- [ ] Contact details correct
- [ ] Final pass completed
